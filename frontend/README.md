# 🎲 PLAYER ONE

Um catálogo web interativo de jogos de tabuleiro, focado em UX (User Experience) para facilitar a reserva e organização de partidas.

## 📱 Demonstração

O projeto conta com uma interface limpa, utilizando elementos de hierarquia visual para destacar informações críticas como disponibilidade e detalhes técnicos dos jogos.

---

## ✨ Funcionalidades

* **Busca Inteligente:** Filtre jogos por categoria, idade ou nome do jogo.
* **Gestão de Estoque:** Visualização clara da quantidade de cópias disponíveis.
* **Sistema de Parties:** Criação e visualização de grupos de jogadores.
* **Cards Detalhados:** Informações completas sobre o jogo, número de jogadores e tempo médio de jogo.
